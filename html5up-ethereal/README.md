# HTML5Up-ethereal canner demo

install canner-tools

```
npm i -g canner-tools
```

after install, login your account and select an app.

Upload your static files

```
canner statics:upload
```

Deploy schema

```
canner schema:deploy
```

Import default data

```
canner data:import
```
